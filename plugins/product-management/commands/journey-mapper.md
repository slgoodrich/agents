# Journey Mapper

Create customer journey maps showing user touchpoints, emotions, and opportunities.

## Usage

```
/pm:journey-map [user persona or scenario]
```

## What This Command Does

Maps the end-to-end customer journey including stages, actions, thoughts, emotions, pain points, and opportunities.

## Instructions

1. Identify journey stages (Awareness → Consideration → Purchase → Usage → Advocacy)
2. For each stage map:
   - User actions
   - Thoughts and questions
   - Emotional state
   - Touchpoints
   - Pain points
   - Opportunities for improvement

## Template

```markdown
# Customer Journey Map: [Persona/Scenario]

## Stage 1: [Awareness]

**Actions**: [What user does]
**Thoughts**: [What they're thinking]
**Emotions**: 😐 [Neutral/curious]
**Touchpoints**: [Where they interact]
**Pain Points**: [Frustrations]
```
