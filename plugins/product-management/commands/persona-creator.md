# Persona Creator

Generate detailed user personas from research, data, and customer insights.

## Usage

```
/pm:persona [target user segment or role]
```

## What This Command Does

Creates research-based user personas including demographics, goals, pain points, behaviors, and needs.

## Instructions

1. Ask for research inputs or generate from description
2. Create persona with:
   - Name and photo (stock image description)
   - Demographics and background
   - Goals and motivations
   - Pain points and frustrations
   - Behaviors and preferences
   - Quotes and scenarios
   - Technology usage
   - Product needs

## Template

```markdown
# User Persona: [Name]

![Photo description]

## Profile
- **Age**: [Range]
- **Role**: [Job title]
- **Company**: [Size/industry]
- **Location**: [Region]
- **Experience**: [Years in role]

## Quote
"[Representative quote capturing mindset]"

## Goals
- [Primary goal]
- [Secondary goal]
- [Aspirational goal]

## Frustrations
- [Main pain point]
- [Workflow friction]
- [Tool/process frustration]

## Behaviors
- **Daily workflow**: [Typical day]
- **Tools used**: [Current solutions]
- **Decision-making**: [How they choose products]
- **Content consumption**: [Where they learn]

## Needs from Product
- [Must-have capability]
- [Important feature]
- [Nice-to-have]

## Scenario
[Day-in-the-life story showing how they'd use product]
```
