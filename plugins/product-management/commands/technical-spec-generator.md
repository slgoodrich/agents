# Technical Specification Generator

Generate detailed technical specifications for features and systems.

## Usage
```
/pm:technical-spec [feature or system]
```

## What This Does
Creates comprehensive technical specifications including architecture, APIs, data models, security, and performance requirements.

## Output Includes
- System architecture
- API endpoints and contracts
- Data models and schemas
- Security requirements
- Performance criteria
- Integration points
- Testing strategy
