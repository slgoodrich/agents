# Growth Frameworks

## Overview
Models and frameworks for sustainable product growth through acquisition, activation, retention, and expansion.

## AARRR (Pirate Metrics)

### Acquisition
**How do users find you?**
- Channels: Organic, paid, referral, social
- Metrics: Traffic, signups, CAC

### Activation
**Do users have a great first experience?**
- Onboarding completion
- Aha moment
- Time to value

### Retention
**Do users come back?**
- DAU/MAU ratio
- Cohort retention curves
- Churn rate

### Revenue
**How do you monetize?**
- Conversion rate
- ARPU, LTV
- Payment success

### Referral
**Do users tell others?**
- Viral coefficient
- Referral rate
- Net Promoter Score

## Growth Loops

### Loop Structure
1. **Input** - User action
2. **Output** - Value created
3. **Feedback** - Drives new input

### Loop Types

**Viral Loops**
- User invites → New users → More invites
- Example: Dropbox referrals

**Content Loops**
- Users create content → Attracts visitors → New users create content
- Example: YouTube, Medium

**Paid Loops**
- Revenue → Marketing spend → New users → More revenue
- Example: SaaS with healthy CAC:LTV

**Ecosystem Loops**
- More buyers → Attracts sellers → More inventory → More buyers
- Example: Marketplaces

## Growth Models

### Product-Led Growth (PLG)
- Free trial or freemium
- Self-serve onboarding
- Product delivers value before purchase
- Viral distribution

### Sales-Led Growth
- Outbound sales
- High touch onboarding
- Enterprise focus
- Relationship-driven

### Community-Led Growth
- User communities
- User-generated content
- Network effects
- Brand advocates

## North Star Metric
Single metric that best captures core value
- Leading indicator of success
- Reflects customer value
- Drives long-term growth

**Examples**:
- Airbnb: Nights booked
- Slack: Messages sent by teams
- Spotify: Time listening

## Growth Equation
**Understand your levers**
- Active users = New + Retained + Resurrected - Churned
- Revenue = Users × Conversion % × ARPU

## When to Use
Growth strategy, product-market fit, scaling, metric selection
