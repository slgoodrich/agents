# Product Analytics

## Overview
Systematic approach to measuring, analyzing, and optimizing product performance through data-driven insights.

## Analytics Frameworks

### Event-Based Analytics
**Track user actions**
- Event: User action (click, view, submit)
- Properties: Metadata (button_id, page, timestamp)
- User properties: Demographics, segment

### Funnel Analysis
**Conversion optimization**
- Define steps in journey
- Measure drop-off rates
- Identify friction points
- A/B test improvements

### Cohort Analysis
**User behavior over time**
- Group users by common characteristics
- Track behavior across time periods
- Identify retention patterns
- Segment by acquisition channel

### Retention Curves
**Long-term engagement**
- Day 1, 7, 30, 90 retention
- Cohort retention over time
- Identify plateau (retained users)

## Key Metrics by Stage

### Acquisition
- Traffic sources
- Signup conversion rate
- Cost per acquisition (CAC)

### Activation
- Onboarding completion rate
- Time to aha moment
- Feature adoption rate

### Engagement
- Daily/Monthly Active Users (DAU/MAU)
- Session frequency
- Session duration
- Feature usage

### Retention
- Retention rate by cohort
- Churn rate
- Customer lifetime (months)

### Monetization
- Conversion rate (free to paid)
- Average revenue per user (ARPU)
- Lifetime value (LTV)
- LTV:CAC ratio

## Analytics Implementation

### Instrumentation
1. **Define events** - What to track
2. **Event taxonomy** - Naming conventions
3. **Implementation** - SDK integration
4. **Validation** - QA tracking
5. **Documentation** - Data dictionary

### Analysis Methods
- **Descriptive** - What happened
- **Diagnostic** - Why it happened
- **Predictive** - What will happen
- **Prescriptive** - What to do

## Segmentation

### User Segments
- Demographics (age, location)
- Behavioral (power users, at-risk)
- Firmographics (company size, industry)
- Psychographics (goals, motivations)

### Usage Patterns
- Feature adoption
- Frequency of use
- Depth of engagement
- User journey paths

## Data Visualization
- Trend lines (over time)
- Funnels (conversion)
- Cohorts (retention)
- Heatmaps (engagement)
- Distribution (histograms)

## When to Use
Product optimization, feature prioritization, growth initiatives, user research validation
